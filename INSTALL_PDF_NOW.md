# ⚠️ Установка зависимостей для PDF

Для работы экспорта PDF необходимо установить зависимости:

## Windows (PowerShell или CMD):

```bash
cd frontend
npm install jspdf html2canvas
```

## Или используйте batch файл:

Запустите `install-pdf-deps.bat` из корневой папки проекта.

## После установки:

Перезапустите dev-сервер:
```bash
cd frontend
npm run dev
```

## Примечание:

Если зависимости не установлены, кнопка экспорта PDF покажет сообщение об ошибке с инструкцией по установке.
