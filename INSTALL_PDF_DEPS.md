# Установка зависимостей для PDF-генерации

Для работы PDF-генерации необходимо установить дополнительные зависимости:

```bash
cd frontend
npm install jspdf html2canvas
```

После установки перезапустите dev-сервер:

```bash
npm run dev
```

## Использование

PDF-генерация доступна на странице "Статистика" через кнопку "Экспорт PDF".
